## Installation & Setup

### Option 1: Run with Live Server
1. Open project folder in VS Code
2. Right-click `index.html`
3. Select **Open with Live Server**

### Option 2: Run with XAMPP
1. Copy project folder to:
   ```
   C:/xampp/htdocs/
   ```
2. Start Apache in XAMPP
3. Open browser and visit:
   ```
   http://localhost/Jobsy
   ```

---

## API Configuration
1. Create an account on RapidAPI
2. Subscribe to a Job Search API
3. Open `script.js`
4. Replace the API key:
```javascript
const API_KEY = "YOUR_RAPIDAPI_KEY";
```
